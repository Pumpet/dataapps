Для установки необходимо наличие MSSQL LocalDB версия не ниже 2014.
Базу, конечно же, можно прицепить и к полноценному MSSQL.
При необходимости установите .NET Framework 4.6.1.

Установки не требуется. 
Содержимое каталога stable скопируйте в любой каталог и запускайте LabGen.exe
Перед запуском исправьте пути в LabGen.xml
При необходимости поправьте в ConnectionString наименование сервера.
FormsLabGen.xml можно удалить, тогда формы примут размеры и расположение по умолчанию.
